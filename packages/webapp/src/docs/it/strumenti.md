# Strumenti

Il menu **Strumenti** raccoglie utilità che non richiedono un account.

## Genera dati di esempio

Crea un file CSV di prova con colonne e righe casuali: utile per esplorare l'editor senza avere ancora dati veri. Scarichi il file e lo carichi in un grafico.

## Genera punti su mappa

Crea un dataset di coordinate (latitudine e longitudine) sul territorio italiano, pronto per una mappa a punti.

## Preview dei file GeoJSON

Carica un file GeoJSON e verificane il contenuto su una mappa prima di usarlo: confini, geometrie e proprietà.

## Esempi

La voce **Esempi** della barra di navigazione mostra grafici pronti costruiti con dati di esempio: un buon punto di partenza per capire cosa si può ottenere.
